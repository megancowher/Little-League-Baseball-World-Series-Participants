# Little-League-Baseball-World-Series-Participants
Notes on the raw data:
- Data was pulled from the Little League International website and supplemented by Wikipedia. Where discrepancies existed, the Little League website was considered the more trustworthy site.
- Data does not exist for the 2020 tournament as it was canceled due to COVID-19.
- Data is not yet in for the 2023 tournament as it is currently underway and winners have not been determined.
